# Workshop-Build-a-Caesar-Cipher-Instructions
Workshop do Freecodecamp (Certificação em Python), com as praticas em como escrever um script da cifra de Cesar (em python) .

É uma das técnicas mais simples para criptografar texto, consistindo em substituir cada letra do texto claro pela letra encontrada a uma quantidade fixa de posições à frente no alfabeto

* Workshop.py contém o conteúdo que foi sendo modificado conforme as instruções (de forma gradual, conforme à ordem das solicitações).
* FinalCode.py contém o código final em python do Workshop (de forma rígida, para passar no teste).
* CodigoNovo.py contém o código atual com minhas modificações.
